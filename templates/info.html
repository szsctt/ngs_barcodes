Barcode information

Add all barcode information on this page. Barcodes are arranged into sets which occur at the same place in the read. A set of barcodes can be 'variable' or 'constant'. A 'constant' set of barcodes consists of the set of all sequences that can be found at a specified place in the read. All barcodes in a constant set must have the same length. A 'variable' set of barcodes consists of all the sequences found between the 'before' sequence and the 'after' sequence. These 'barcodes' may be of any length.
